#  PROJETOS

---

Os projetos desta seção foram criados para apresentar o Itaquerino de forma prática, clara e acessível. Cada atividade explora um conceito fundamental 
de eletrônica ou programação, sempre com foco em entendimento real — não apenas em “fazer funcionar”.
Aqui você encontra tutoriais curtos, replicáveis em sala de aula e ideais para quem está começando no mundo dos microcontroladores. Cada projeto inclui 
materiais, esquema, código e explicações sobre o que está acontecendo por trás do circuito, para garantir que o aluno compreenda o princípio e não só 
repita passos.
